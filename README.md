# Real Estate Listing Automation (Web Scraping + Selenium)

An end-to-end automation project that scrapes real estate listings from a static website and automatically submits the extracted data into a Google Form, generating a structured Google Sheet of listings.

This project demonstrates practical web scraping, browser automation, and real-world data pipeline logic — without using any APIs.

---

## What This Project Does

1. Scrapes real estate listings from a static website using **BeautifulSoup**
2. Extracts:
   - Property price
   - Property address
   - Listing link
3. Stores the extracted data in separate Python lists
4. Automatically opens a Google Form using **Selenium WebDriver**
5. Fills and submits the form for **each listing**
6. Repeats the process until all listings are submitted
7. Outputs a clean, structured **Google Sheet** generated by the form

## How to Run the Project
1. Clone the repository
```bash
git clone https://github.com/your-username/real-estate-automation.git
```
2. Install dependencies
```bash
pip install requests beautifulsoup4 lxml selenium
```
3. Set up Selenium WebDriver

- Install Chrome
- Download matching ChromeDriver
- Ensure it is accessible via PATH

4. Update URLs

In main.py, replace:
- Google Form URL
- Form field selectors (if needed)

5. Run the script
```bash
python main.py
```
